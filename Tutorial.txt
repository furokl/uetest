Сцена
-------------------------------------------------------------------------------
Камера
	ПКМ						- вращение
	ПКМ + W/A/S/D			- перемещение
	Ролик					- изменение скорости
	F						- фокус
	G						- спрятать значки
	alt + ЛКМ				- вращение вокруг точки
	alt + ПКМ				- зум вокруг точки
	ctrl(правый) + L + ЛКМ	- управлять глобальным светилом
	shift + transparent		- переместить объект с фиксацией камеры за объектом
	F11						- fullscreen

Инструментарий
	Q -	курсор
	W - transparent
	E - rotate
	R - scale

Объект
	ctrl + c - ctrl + v			- дублировать
	ctrl + d					- дублировать (с небольшим смещением)
	alt + потянуть в сторону	- дублировать

Контент браузер (нижняя панель)
	ctrl + пробел	- открыть

Хуйня справа-сверху
	Gizmo:
		Переключение глобальные/локальные координаты
	Snapping:
		Шаг transparent
		Шаг rotation
		Шаг Scale

Неприятности
	Alt + F		- Черное небо (Отключение атмосферы)

Blueprints
-------------------------------------------------------------------------------
Управление
	1 + ЛКМ		- создать const
	2 + ЛКМ		- создать const Vector2
	3 + ЛКМ		- создать const Vector3
	4 + ЛКМ		- создать const Vector4
	S + ЛКМ		- создать const сразу как параметр
	M + ЛКМ		- создать multiply
	alt + ЛКМ	- удалить нод

Нода
	ПКМ
		Convert To Parametr - установить как параметр
			(можно будет менять из вкладки details и в свойстве нода)
